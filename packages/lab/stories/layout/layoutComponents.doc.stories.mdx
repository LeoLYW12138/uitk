# Layout Components

Layout componentry was created to help you build consistent and responsive layouts while using Toolkit components.

## Border Layout

Defines the main content regions of an application, region or widget, such as a the footer, header or side navigation.

The region is defined by the `position` prop.

<Canvas>
  <Story name="Border Layout">
    <BorderLayout>
      <BorderItem
        position="header"
        className="item"
        style={{ backgroundColor: "#c5afa4" }}
      >
        <p>Header</p>
      </BorderItem>
      <BorderItem
        position="left"
        className="item"
        style={{ backgroundColor: "#cc7e85" }}
      >
        <p>Left</p>
      </BorderItem>
      <BorderItem
        position="main"
        className="item"
        style={{ backgroundColor: "#cf4d6f", minWidth: 100 }}
      >
        <p>Main</p>
      </BorderItem>
      <BorderItem
        position="right"
        className="item"
        style={{ backgroundColor: "#a36d90" }}
      >
        <p>Right</p>
      </BorderItem>
      <BorderItem
        position="bottom"
        className="item"
        style={{ backgroundColor: "#76818e" }}
      >
        <p>Bottom</p>
      </BorderItem>
    </BorderLayout>
  </Story>
</Canvas>

## Grid Layout

Defines an equally distributed layout system using columns and rows.

It defaults to a twelve column grid but you can customise it using the `columns` and `rows` props.

<Canvas>
  <Story name="Grid Layout">
    <GridLayout className="gridLayout" columnGap="1rem" rowGap="1rem">
      {Array.from({ length: 12 }, (_, index) => (
        <GridItem className="item" style={{ backgroundColor: "lightcyan" }}>
          <p>{`GridItem ${index + 1}`}</p>
        </GridItem>
      ))}
    </GridLayout>
  </Story>
</Canvas>

## Parent Child Layout

Displays a hierarchical structure comprising of a main content area and an accompanying ‘parent’ region, used to drive the content that’s displayed.

<Canvas>
  <Story name="Parent Child Layout">
    <ParentChildLayout
      className="flexLayout"
      parentWidth={150}
      parent={
        <div
          className="flexItem"
          style={{
            display: "flex",
            alignItems: "center",
            justifyContent: "center",
            height: 220,
          }}
        >
          <p>Parent</p>
        </div>
      }
      child={
        <div
          className="flexItem"
          style={{
            display: "flex",
            alignItems: "center",
            justifyContent: "center",
            height: 220,
          }}
        >
          <p>Child</p>
        </div>
      }
    />
  </Story>
</Canvas>

## Panelled Layout

Provides multiple content zones with flexible heights and widths.

It uses the region splitter by default to allow resizing of content areas.

<Canvas>
  <Story name="Panelled Layout">
    <PanelledLayout className="panelledLayout">
      {Array.from({ length: 4 }, (_, index) => (
        <div className="panelledItem" key={index}>
          <p>{`Panel ${index + 1}`}</p>
        </div>
      ))}
    </PanelledLayout>
  </Story>
</Canvas>
